# B站图片转评论

[前端](# "前端") | [后端](# "后端")
### #项目介绍
Bilibili图片转评论后台项目源码分享，功能可以将B站视频下方的评论1:1还原成免扣可直接使用的png图片。

------------
`Python3.8`

`Flask1.1.2`

`aiohttp3.6.2`

------------
